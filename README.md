# OIBSIP_Python_Task3 - Basic Weather App

A simple **Python-based Weather App** that fetches and displays real-time weather information for any city using the **OpenWeatherMap API**.

## Features
- Enter any city name to get its current weather
- Displays:
   - City name
  - Temperature (in Celsius)
  - Humidity
  - Weather condition (description)
  - Handles invalid city names and network errors gracefully

- Python 3.x
- `requests` library
